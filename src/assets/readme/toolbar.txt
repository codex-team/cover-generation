Класс Toolbar

Параметры конструктора:
{Element} elem        - DOM-элемент которому привызывается объект
{Element} canvas      - DOM-элемент (пока нет класса canvas, потом будет объект этого класса) холста
{Object} style        - объект с дополнительными свойствами
{
    {Number} padding  - ширина отступа между elem и toolbar'ом
}

Приватные поля:
{null} controllable - ссылка на DOM-элемент elem
{null} toolbar      - ссылка на DOM-элемент toolbar'a 

Приватные методы:
{null} changeToolbarPosition({null})             - передвижение toolbar к elem
{null} hideToolbar({Event} event)                - сокрытие toolbar при сведении фокуса с elem
{null} showToolbar({Event} event)                - показ toolbar при двойном клике на elem
{null} changeControllableColor({Event} event)    - смена цвета элемента при клике на соответствующую иконку в режиме выбора цвета
{null} toggleToolbarColorMode({Event} event)     - переход toolbar'a в режим выбора цвета
{null} changeControllableFontSize({Event} event) - смена размера шрифта elem
{null} changeControllableAlign({Event} event)    - смена положения elem
